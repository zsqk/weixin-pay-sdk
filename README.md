# weixin-pay-sdk

weixin pay SDK for deno (v3)

适用于微信支付 API 的 JS SDK. 使用 TypeScript 编写, 推荐在 Deno 环境下运行.

主要特性为:

1. 功能. 计划支持微信支付 API 的所有功能.
2. 性能. 尽量使用 Web API, 而不是 JS, 保证性能.
3. 兼容. 尽量使用 Web API, 兼容浏览器环境及 Deno.
